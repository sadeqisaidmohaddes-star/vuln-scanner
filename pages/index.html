vulnscan

A modular asynchronous scanner for defensive network assessments and read-only source-repository analysis, with console, JSON, HTML, and browser dashboard reporting.

Start locally

Requires Python 3.11+. The authorization flag is deliberate: it makes scan intent explicit before any assessment begins.

git clone https://github.com/sadeqisaidmohaddes-star/vuln-scanner.git
cd vuln-scanner
python -m venv .venv
pip install -r requirements.txt
python -m vulnscan --list-modules
